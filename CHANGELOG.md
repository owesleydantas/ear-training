## [1.0.0] – 2026-01-09

### Added
- Treinamento auditivo de tríades (maior, menor, aumentado, diminuto)
- Reprodução de acordes por síntese
- Sistema de score (acertos e erros)
- Opção de variação de fundamental (notas naturais)
- Repetição controlada do acorde atual
- Interface simples e funcional em português
- Deploy via GitHub Pages com Vite
